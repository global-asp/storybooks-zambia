Ngwahase kututa nacho.

I can run with it.

Kuti nabutuka nao

Ngwahase kutumbuka nacho.

I can jump with it.

Kuti natoloka nao

Ngwahase kukina nacho.

I can dance with it.

Kuti nasha nao

Ngwahase kusana nacho.

I can swim with it.

Kuti na owa nao

Ngwahase kulivuwa nacho.

I can skip with it.

Kuti naciluka nao

Ngwahase kwasa nacho.

I can kick with it.

Kuti napanta nao

Ngwahase kuchina nacho.

I can escape with it.

Kuti naluba nao

Unonyi kangweshi kuhasa kumbululuka nachoko.

But I can never fly with it.

Teti impupuke nao

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Masho Kaloza
Language: Chiluvale
